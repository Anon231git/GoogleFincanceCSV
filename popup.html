CSV Portfolio Importer

How to Use:

  1. Open your Google Finance portfolio page
  2. The CSV uploader will appear in the top-right corner
  3. Upload a CSV file with stock symbols
  4. Configure settings and click "Add Stocks to Portfolio"

CSV Format:

Symbol,Name
AAPL,Apple Inc
GOOGL,Alphabet Inc
MSFT,Microsoft
TSLA,Tesla Inc
Note: The extension will automatically detect columns with headers like "symbol", "stock", "name", or "ticker". If no headers are found, it will use the first column.
Please navigate to a Google Finance portfolio page to use this extension.
